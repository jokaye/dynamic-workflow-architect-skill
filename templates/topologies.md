# Dynamic Workflow Topologies

## Map-reduce audit
Inventory -> Shard -> Map reviewers -> Normalize -> Reduce -> Cross-check -> Report

## Staged migration
Recon -> Plan rules -> Pilot -> Fan-out edits -> Integrate -> Verify -> Repair loop -> Report

## Cross-checked research
Decompose -> Source search -> Claim extraction -> Independent verification -> Conflict resolution -> Cited report

## Multi-angle architecture review
Constraints -> Candidate plans -> Adversarial critique -> Tradeoff matrix -> Recommendation

## Test-generation campaign
Surface inventory -> Risk ranking -> Test generation -> Targeted test run -> Repair -> Coverage/risk report
